# سند - منصة الأتمتة المالية الذكية

💰 منصة تقنية متكاملة تربط بين عمليات الدفع الإلكتروني وأنظمة الجرد والمحاسبة.

## المميزات

- 🤖 بوت المطابقة الآلية
- 🔒 أمان بنكي
- 📱 واتساب مدمج
- 📊 تقارير ذكية
- ⚡ Offline First
- 🏦 ربط مع البنوك الفلسطينية

## التقنيات المستخدمة

- React 18
- CSS Modules
- RTL Support
- Fully Responsive

## التثبيت والتشغيل

```bash
# تثبيت الاعتماديات
npm install

# تشغيل المشروع محلياً
npm start

# بناء المشروع للإنتاج
npm run build
